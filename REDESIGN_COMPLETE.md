## 🛠️ Customization Options

### Change Primary Color:
Edit `public/css/login.css`:
```css
/* Change #3b82f6 to your brand color throughout the file */
background: linear-gradient(135deg, #3b82f6 0%, #2563eb 100%);
```

### Adjust Animations:
```css
/* Speed up/slow down gear rotation */
.gear-animation {
    animation: rotate-gear 30s linear infinite; /* Change 30s */
}

/* Adjust grid scroll */
.grid-pattern {
    animation: grid-scroll 20s linear infinite; /* Change 20s */
}
```

### Modify Logo Size:
```css
.brand-logo {
    max-width: 160px; /* Adjust width */
    max-height: 160px; /* Adjust height */
}
```
